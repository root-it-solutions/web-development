== Description ==
A school and education WordPress theme that ships with dedicated content blocks for featuring your classes, teachers and news.The theme comes with an interactive class or activity timetable.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
